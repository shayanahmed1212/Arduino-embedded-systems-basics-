## 📌 &nbsp;Repository Purpose

This repository is **intentionally structured to show progressive learning** — from basic syntax and control structures to simple sensor-based mini projects. Rather than one polished app, every sketch is a standalone lesson.

<table>
<tr>
<td width="50%" valign="top">

**The goal of this repo is to:**

- ✅ Practice Arduino fundamentals from scratch
- ✅ Understand logic before reaching for libraries
- ✅ Learn hardware–software interaction step by step
- ✅ Build confidence for larger robotics and IoT systems

</td>
<td width="50%" valign="top">

**These sketches are building blocks for:**

- 🤖 Smart robot cars
- 🌐 IoT systems using ESP32 / ESP8266
- 📡 Sensor-driven automation pipelines
- 🔩 Real-time embedded control systems

</tr>

## 💡 &nbsp;Learning Approach

Instead of jumping into complexity, this repository follows a **deliberate bottom-up strategy:**

```
  [ Single Concept Sketch ]
           │
           ▼
  [ Test on Hardware ]
           │
           ▼
  [ Understand Pin Behaviour + Timing ]
           │
           ▼
  [ Combine with Next Concept ]
           │
           ▼
  [ Build Mini Project ]
```

Each sketch is small, readable, and focused on **one idea at a time** — no bloated libraries, no magic, just logic.
