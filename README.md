# Arduino Basics & Mini Projects

A collection of learning-focused Arduino sketches created to understand core
embedded systems and microcontroller programming concepts. This repository is
intentionally structured to show **progressive learning**, from basic syntax and
control structures to simple sensor-based mini projects.

---

## 📌 Repository Purpose

This repository serves as a **foundation-level learning archive** rather than a
single polished project. Each sketch focuses on one concept at a time, helping
build a strong understanding of how Arduino hardware and software interact.

The goal of this repository is to:
- Practice Arduino fundamentals
- Understand logic before using advanced libraries
- Learn hardware–software interaction step by step
- Build confidence for larger robotics and IoT projects

---

## 🧠 Topics Covered

- Digital input/output
- Analog input/output
- Serial communication
- Control structures (if-else, loops, switch-case)
- Creating and using functions
- LED control and brightness variation
- Ultrasonic sensor basics
- Sensor and actuator integration (LED, buzzer)

---

## 🛠️ Hardware Used

- Arduino Uno
- LEDs
- Resistors
- Ultrasonic Sensor (HC-SR04)
- Buzzer
- Jumper wires
- Breadboard

---

## 💡 Learning Approach

Instead of jumping directly into complex projects, this repository focuses on:
- Writing small, understandable sketches
- Testing one concept at a time
- Building logic from scratch
- Understanding how Arduino pins, timing, and conditions work

These sketches act as **building blocks** for more advanced projects such as:
- Smart robot cars
- IoT systems using ESP32 / ESP8266
- Sensor-driven automation
